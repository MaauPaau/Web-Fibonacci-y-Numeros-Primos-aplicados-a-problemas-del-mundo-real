Serie de Fibonacci

En la naturaleza, las plantas organizan sus hojas, pétalos y semillas siguiendo la serie de Fibonacci (0, 1, 1, 2, 3, 5, 8…). Los girasoles tienen 34 y 55 espirales de semillas — ambos números de Fibonacci. Este patrón optimiza el espacio y la captación de luz.

a = 0, b = 1
c = a + b → nuevo término
a = b, b = c → repetir
🔐

Números Primos

Un número primo solo es divisible entre 1 y sí mismo. Son la base de la criptografía moderna (RSA, HTTPS). Algunos términos de Fibonacci también son primos — llamados Fibonacci primos — y son especialmente raros y valiosos.

Para cada número n:
contador = 0
si divisores == 2 → es primo
